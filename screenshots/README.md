# Hospital Management System (Java Swing)

A desktop-based Hospital Management System developed using Java and Swing.
The application provides a graphical interface to manage hospital operations
such as patients, doctors, appointments, and billing.

## Features
- Add and view patient details
- Add and view doctor details
- Manage appointments
- Generate and view billing information

## Technologies Used
- Java
- Swing (JFrame, JButton, JTable, JOptionPane)
